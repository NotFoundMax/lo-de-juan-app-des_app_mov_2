# Glosario — Lo de Juan

Términos del negocio y del sistema usados en la documentación.

## Negocio

| Término | Significado |
|---|---|
| **Pollería** | Restaurante especializado en pollos a la brasa y parrilla |
| **Pollería al spiedo** | Pollo cocinado en un asador rotatorio (fuego directo) |
| **Parrilla** | Porciones de carne/pollo cocinadas en parrilla a carbón |
| **Mostrador** | Atención al cliente en el local (pedidos para llevar o consumo en local) |
| **Delivery** | Reparto a domicilio por repartidores propios |
| **Recogida** | Cliente pide online y retira en el local |
| **Vuelto** | Dinero que se devuelve al cliente cuando paga con monto mayor al total |
| **POS** | Punto de venta (Point of Sale) — terminal para cobrar en mostrador |
| **Ticket promedio** | Gasto promedio por cliente por visita |
| **Rotación de stock** | Frecuencia con que se venden los productos (ventas / inventario promedio) |
| **Stock mínimo** | Cantidad por debajo de la cual se muestra alerta de reabastecimiento |
| **Platillo** | Producto individual del menú (Pollo entero, ½ pollo, Parrilla mixta, etc.) |
| **Menú** | Conjunto de platillos ofrecidos (carta) |
| **Comanda** | Pedido ingresado al sistema que va a la cocina |
| **Fidelización** | Programa de puntos para incentivar compras recurrentes |
| **Canje** | Intercambio de puntos por productos o descuentos |

## Sistema

| Término | Significado |
|---|---|
| **Auth** | Sistema de autenticación y registro de usuarios |
| **Realtime** | Suscripciones en vivo a cambios en la base de datos (notificaciones push) |
| **Storage** | Almacenamiento de archivos (imágenes de productos, PDFs de capacitación) |
| **Trigger** | Función automática que se ejecuta ante un evento en la BD |
| **Índice compuesto** | Índice sobre múltiples campos para consultas rápidas en Firestore |
| **ADR** | Architecture Decision Record — registro de una decisión técnica y su fundamento |
| **Mermaid** | Sintaxis para diagramas (código → gráfico) que GitHub renderiza nativo |
| **Yape** | App de pagos móviles del BCP (más usada en Perú) |
| **Plin** | App de pagos móviles interoperable entre bancos peruanos |
| **Pasarela de pagos** | Servicio que procesa pagos con tarjeta (Visa, Mastercard) |
| **Seed** | Datos de ejemplo precargados para desarrollo/pruebas |
| **RBAC** | Role-Based Access Control — control de acceso por roles |
