# Guía de setup — Cliente (usuario final)

Guía para clientes que quieren usar la app para hacer pedidos.

## Requisitos

- **Expo Go** (Android / iOS) o acceso vía web
- Datos móviles o WiFi

## 1. Abrir la app

El establecimiento tiene un **código QR** visible en el local o en sus redes sociales.

| Dispositivo | Acción |
|---|---|
| **Android** | Abrí Expo Go → escaneá el QR |
| **iPhone** | Abrí Expo Go → escaneá el QR |
| **Web** | Ingresá al enlace que te proporciona el local |

## 2. Crear cuenta

1. Tocá **Registrarse**
2. Ingresá:
   - Nombre
   - Email
   - Teléfono
   - Contraseña
3. Confirmá el registro

> También podés navegar el catálogo sin registrarte, pero para pedir necesitás una cuenta.

## 3. Hacer un pedido

1. Navegá el **catálogo** de productos
2. Tocá un producto para ver detalle
3. Elegí cantidad y **Agregar al carrito**
4. Andá al carrito y **Confirmar pedido**
5. Elegí:
   - **Delivery** → ingresá tu dirección
   - **Recogida en local** → pasás a buscarlo
6. Elegí método de pago
7. ¡Pedido confirmado! ✅

## 4. Seguir tu pedido

Después de confirmar, ves el estado en tiempo real:

| Estado | Significado |
|---|---|
| 🟡 Pendiente | El local recibió tu pedido |
| 🔵 En preparación | Están cocinando 🍳 |
| 🟢 Listo | Podés pasar a recoger / está en camino |
| ✅ Entregado | Recibido |

Si elegiste delivery, recibís una **notificación con la hora estimada** de llegada.

## 5. Perfil y puntos

En tu perfil podés:

- Ver tu **historial de pedidos**
- Ver tus **puntos acumulados** (1 punto por S/.10 gastados)
- **Canjear puntos** por productos
- Cambiar tu contraseña
- Contactar **soporte**

## 6. Problemas frecuentes

| Problema | Solución |
|---|---|
| No puedo registrarme | Verificar que el email sea válido |
| No veo productos en el catálogo | El local puede estar cerrado |
| Error al pagar | Probar otro método de pago |
| No llega la notificación | Revisar que las notificaciones estén activadas |
